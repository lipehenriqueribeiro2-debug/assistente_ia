Atue como Arquiteto de Software React/Next.js. Desenvolva o frontend do AURA_OS com foco absoluto em performance, integrando o DOM ao React Three Fiber.

STACK E ARQUITETURA:
- React + Tailwind CSS + Framer Motion (obrigatório para orquestração de UI).
- O Canvas 3D deve ficar isolado de re-renderizações do React. Use `useRef` e `useMemo` rigorosamente.
- Layout: O HTML não pode ter fundos opacos gigantes. O `<main>` envolve o 3D, e o `<Header>` (fixed z-50) e o `<OverlayUI>` (absolute z-40) flutuam por cima.

FUNCIONALIDADES COREOGRAFADAS:
- Bento Grids (Apresentação de Dados): Devem implementar Staggered Animation no Framer Motion. Os elementos internos (textos, gráficos de barra) animam em cascata.
- Layout Morphing (Transição de Tela): Abandone a troca de páginas tradicional. A transição da Visão Orbital para a Tela de Chat deve usar `<AnimatePresence mode="wait">`. 
- Animação de Saída (Awwwards Style): O texto do ambiente orbital não apenas some; ele usa física de mola (Spring) para quebrar as linhas e voar para fora do eixo Y negativo independentemente, limpando o palco antes do chat se materializar.
